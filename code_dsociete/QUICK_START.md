### Add Your Domain (www.dquantsociete.com)
1. Netlify dashboard → **Domain Settings**
2. Click **Add custom domain**
3. Enter: `dquantsociete.com`
4. Copy provided DNS records
5. Go to Cafe24 → DNS Settings
6. Add records:
   - **A Record**: `@` → `75.2.60.5`
   - **CNAME**: `www` → `[your-site].netlify.app`
7. Wait 10-30 minutes

**HTTPS enabled automatically!** 🔒

## 🎨 CUSTOMIZATION BASICS

### Change Colors
Open `index.html`, find `tailwind.config`, modify:
```javascript
institutional: {
  blue: '#002045',    // Your primary color
  slate: '#4A5568',   // Your secondary color
  gold: '#C5A572'     // Your accent color
}
```

### Update Slider Images
Find these lines in `index.html`:
```html
<div class="slide" style="background-image: url('YOUR_IMAGE_URL');">
```
Replace `YOUR_IMAGE_URL` with your image URLs.

### Edit Content
All text is inside `index.html`. Search for section IDs:
- `#about` → About section
- `#governance` → Leadership section
- `#strategy` → Strategy section
- `#social-mission` → Social mission section
- `#contact` → Contact section

## 🔍 TROUBLESHOOTING

### Images Don't Load
**Problem**: Unsplash URLs blocked  
**Solution**: Replace with your own image URLs

### Slider Doesn't Auto-Advance
**Problem**: JavaScript not loaded  
**Solution**: Check console (F12), ensure no errors

### Animations Not Smooth
**Problem**: Low-end device  
**Solution**: Normal — will work better on production with optimized images

### Mobile Menu Not Working
**Problem**: Placeholder function  
**Solution**: Shows alert — implement full menu in future version
